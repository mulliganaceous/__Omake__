Non-Java Browsers

Problem: When you preview your web page in a browser, the page contains a message reading “Sorry, this page requires a Java-capable browser.”

Cause: If you have an old web browser, it may not support the Java language and, therefore, won’t work with JavaSketchpad. Contact your browser manufacturer for a newer version. Alternately, it may be that your browser supports Java, but that it’s currently set to disable Java applets. Go to your browser’s Preferences or Options to turn on support for Java applets.
