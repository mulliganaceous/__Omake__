What's New

The Geometer's Sketchpad Version 4 has several new features that were added after the original printing of the Reference Manual. In addition, several new features and improvements have been added since the original 4.00 Release. For details, see Changes Since Release 4.00.

If you've used Sketchpad Version 3, you may want a quick overview of the many new features of Version 4, as well as notes on important changes in how you use Sketchpad. For details of the changes, see If You've Used Version 3 and have a look at the Sample Documents which illustrate Sketchpad's features.
