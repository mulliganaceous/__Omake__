Creating a Tool Folder

You can create a folder in which you store documents containing frequently used tools. These tools will be available each time you use Sketchpad.

1.  Locate the folder the Sketchpad program is located in.

2.  If there is no folder named Tool Folder within the folder containing Sketchpad, create a new folder.

3.  Name this new folder Tool Folder.

You can now use this folder to store Sketchpad documents containing frequently used tools. The tools will be available the next time you start Sketchpad.

If you do not have permission to write to the folder containing Sketchpad, it's possible to create a tool folder in an alternate location. See Alternate Tool Folders for details.
