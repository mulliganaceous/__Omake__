Showing or Hiding the Script View

Choose Show Script View or Hide Script View from the Custom Tools menu to show or hide the script view of the active tool. This view allows you to see the given objects and the steps the tool takes to construct its results, as well as, to change the properties of the steps, and to observe and control the tool as it functions.
