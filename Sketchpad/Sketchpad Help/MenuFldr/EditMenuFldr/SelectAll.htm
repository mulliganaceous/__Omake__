Select All

This command selects all objects that match the active tool in the Toolbox.

The keyboard shortcut for Select All is Ctrl+A (Windows) or z+A (Mac).

If a Selection Arrow or Custom tool is active, this command selects all objects in the sketch.

If another tool from the Toolbox is active (Point, Compass, Segment, Ray, Line, or Text), all matching objects are selected. For example, if the Ray tool is active, the command becomes Select All Rays.
