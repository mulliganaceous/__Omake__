Plot Properties

Only loci and function plots have a Plot panel. Both panels allow you to set the number of samples used to plot the locus or function. The greater the number of samples, the smoother and more accurate the plot is. However, the greater the number of samples, the slower Sketchpad can be in calculating and displaying the plot.

The default number of samples assigned to newly constructed loci and function plots is determined by Sampling Preferences.
