Parallel Line

Selection prerequisites: A straight object and one or more points; or a point and one or more straight objects.

Constructs a line through each selected point parallel to each selected straight object.

 

One point, one straight object

One parallel line

 

 

 

Two points, one straight object

Parallel lines through both points

 

 

 

 

One point, two straight objects

Lines parallel to both straight objects
