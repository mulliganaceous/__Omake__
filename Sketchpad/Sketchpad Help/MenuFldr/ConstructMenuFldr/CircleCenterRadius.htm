Circle By Center+Radius

Selection prerequisites: A point and one or more segments and/or distance measurements; or a segment or distance measurement and one or more points.

Constructs one or more circles centered at each selected point and with the radius determined by each selected segment or distance measurement.

One point, one segment

 

One circle

 

Two points, one distance measurement

Two circles centered at the points with radius determined by the measurement

One point, two segments

Two circles centered at the point with radii determined by the segments

This command duplicates the function of a noncollapsible compass, which allows you to set a length, then draw circles centered anywhere with that length as the radius.
