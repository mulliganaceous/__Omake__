Tabulate

This command creates a new table of the selected measurements’ values. Sketchpad’s tables contain entries describing a set of measurements’ values at different points in time. In addition to measured values, you can also tabulate parameters, calculations, measured equations, functions, and composite captions.

This command is available when the selection contains only objects that can be tabulated.

By default, new tables contain one row that dynamically tracks the current values of all tabulated measurements. Values in this row change as measured objects are dragged or animated. You can turn off dynamically-changing values by using the table’s Table Properties panel.
